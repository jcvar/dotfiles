# .dotfiles

**Setup**
1. Clone repo:
    ```
    $ git clone https://github.com/jcvar/dotfiles.git ~/code/.dotfiles
    ```
2. Make symlinks:
    ```
    $ . ~/code/.dotfiles/ln-s.sh
    ```
3. Restart shell or load `.bash_profile`:
    ```
    $ . ~/code/.dotfiles/.bash_profile
    ```
